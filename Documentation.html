Introduction

This project is basicaly demonstration of my practical skills in Markup Languages subject at the University of Life Sciences in Prague. As an example I choose simple database for evidence of employees for some choosen business. In this databse there are many records about some imaginary employees which are divided into 5 segments (Personal data, Sensitive personal data, Contact information, Work data). This is just demonstration, so I have only 5 records inside.

XmlPart.xml
XML document which is basicaly structure.
XsdPart.xsd
XSD document which describes structure of the previous XML document.
Transformation.xslt
XSLT document (eXtensible Stylesheet Language Transformations), Prescription in the recommended styling language for XML documents
HtmlOutput.html
HTML document after applying the above mentioned XSLT transformation
Documentation.html
Documentation about the project.

Methodology

In the project I used few online tools, which make working with markup languages little bit easier.For example I validated html and XML.I also used XSD generator. Down below is a list of used tools.

  1. XML Validator
  2. HTML Validator
  3. XSD/XML Schema Generator
  4. XML Validator - XML Schema (XSD)
  5. XSL Transformer - XSLT

Work content

In the project there are 5 main sections of records about particular employee

  1. Personal data (Basic personal data information)
  2. Sensitive personal data (Basic sensitive, personal data information)
  3. Contact information (Basic contact information)
  4. Work data (Data about work)
  5. Employee Account (Data about user's account)
